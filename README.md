# MySamsat

Aplikasi web untuk merekap transaksi harian pengurusan STNK dengan backend Supabase.

## Fitur Utama
- **Dashboard**: Statistik pendapatan harian, bulanan, dan tren transaksi.
- **Manajemen Transaksi**: Input data STNK, perhitungan otomatis (Pajak + Fee), dan upload lampiran.
- **Laporan**: Rekapitulasi data untuk kebutuhan bulanan dan tahunan.
- **Dark Mode**: Dukungan tema gelap dan terang yang nyaman di mata.
- **Keamanan**: Sistem login terintegrasi dengan Supabase Auth.

## Teknologi
- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion.
- **UI Components**: shadcn/ui, Lucide Icons.
- **Charts**: Recharts.
- **Backend**: Supabase (Auth, Database, Storage).
